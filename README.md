# Conversational AI Agent — Google AI Studio

## Overview
Learning implementation exploring conversational AI agent design using 
Google AI Studio and Gemini 2.5 Flash.

## What I built
- Designed system instructions for a customer-facing chatbot
- Applied metaprompting to generate and refine agent behaviour
- Defined agent persona, knowledge scope, and constraints
- Tested conversational flows including edge cases and out-of-scope queries

## Use Case
Cymbal Manufacturing customer service agent — handling product enquiries, 
custom machining requests, and basic troubleshooting.

## Key Concepts Explored
- System instruction design for LLM-based agents
- Metaprompting techniques
- Agent persona and constraint definition
- Relationship between Google AI Studio prototyping and 
  production Dialogflow CX / CES implementations

## Relevance to Production Systems
This prototype demonstrates core concepts behind Google Customer 
Engagement Suite (CES) and Dialogflow CX agent design. Production 
implementations would use CX Agent Studio with adversarial defence, 
webhook integrations, and enterprise security controls.
